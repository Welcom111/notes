# Notes

Небольшое приложение в системном трее для хранения Markdown-заметок.

Доступно храние в локальной папке или через WebDAV. Меняется в настройках по нажатию ⚙

# Установка
Скачкайте последнюю версию программы из [релизов](https://github.com/Welcom111/notes/releases).

Настройки собранной версии сохраняются в `%APPDATA%\QuickNotes`, а пароль WebDAV — в хранилище учётных данных Windows. 

При первом запуске однофайловый EXE может открываться немного дольше, поскольку распаковывает внутренние компоненты во временную папку.
